# 🎨 Perspective Simulator tool - Trying to Draw

This repository hosts the technical engine for the perspective study tool featured on the **Trying to Draw** YouTube channel.

### 🚀 Access the official tool here:
## [https://tryingtodraw.com](https://tryingtodraw.com)

---

### About the Project
This simulator was developed to assist art students (especially those tackling the **Drawabox** challenge) in visualizing vanishing points and 3D volume construction.

* **YouTube Channel:** [Watch here](https://www.youtube.com/@TryingToDraw) 
* **Goal:** Documenting the journey of learning how to draw from scratch.
